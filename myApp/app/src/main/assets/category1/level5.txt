I java sparas värden i variabler. Dessa variabler har en deklarerad typ. Det finns 8 primitiva typer, där de vanligaste är boolean, int, double och char. booleans kan hålla två olika värden, true och false. double och int håller båda tal, där double håller decimaltal och int håller heltal. char kan hålla ett tecken. Exempel på hur variabler av dessa typer deklareras och instansieras kan ses nedan: row rowint i = 5; rowchar alpha = ‘a’; rowboolean isThisTrue = false; rowdouble decimals = 2.5;row rowDet finns också andra typer i java, en väldigt användbar är String. Denna typen håller strängar av text.
