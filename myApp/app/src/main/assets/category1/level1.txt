[Question]
En variabel kan hålla värden av olika typer. Först skriver man en variabels [1], sen ger man den ett namn. Sist på raden skriver man ett [2] för att avsluta sitt påstående. Detta för att koden ska [3] rätt.
[Answer]
typ,;,kompilera
[Info]
I java, och i många andra språk, använder man variabler för att hålla (spara) olika värden. Dessa värden kan vara av olika typ. En variabel har en typ, mer om det senare, ett namn, och ett värde. När man deklarerar en variabel anger man först dess typ, sen ger man den ett namn med liten bokstav och sen kan man tilldela den ett värde. Tex String text = “Kalles kaviar är bäst”; \n I java finns det en kompilator som kompilerar koden för att se om den går att köra eller inte.  När man är klar med raden måste man därför avsluta med ett ‘;’ för att koden ska kompilera rätt.
[Hint]
Variabler kan hålla olika typer av värden. \n När man är färdig med ett påstående skriver man ett tecken. \n I java finns det en kompilator som kompilerar koden för att se om den går att köra eller inte.
